# 战棋 - 围棋与象棋的融合

一个创新的战略棋盘游戏，融合了围棋的领土控制和象棋的战术对战元素。

## 游戏特色

- **双层次游戏机制**：战略层（围棋棋盘）和战术层（象棋棋盘）的创新融合
- **领土与战术的平衡**：在战略层控制领土，在战术层进行棋子对战
- **实时战斗状态**：动态显示游戏进程和战斗状态
- **战术令牌系统**：通过控制领土获取战术令牌，用于召唤棋子
- **完整的游戏统计**：实时显示步数、游戏时间等统计信息

## 技术栈

- **前端开发**：HTML5, CSS3, JavaScript
- **服务器通信**：Node.js (katago-server.js)
- **AI引擎集成**：KataGo围棋引擎（可选）
- **开发工具**：VSCode

## 安装与运行

### 基本运行

1. 确保您的系统已安装现代浏览器
2. 启动本地HTTP服务器：
   ```bash
   python -m http.server 8001
   ```
3. 在浏览器中访问：`http://localhost:8001`

### 可选：启动KataGo服务器

如需使用AI功能，需要启动KataGo服务器：

1. 确保已安装Node.js
2. 启动服务器：
   ```bash
   node katago-server.js
   ```
3. 服务器将在 `http://localhost:3000` 上运行

## 游戏操作指南

1. **新游戏**：点击"新游戏"按钮开始一局新游戏
2. **悔棋**：点击"悔棋"按钮撤销上一步操作
3. **规则**：点击"规则"按钮查看详细游戏规则
4. **战略层操作**：在围棋棋盘上放置棋子控制领土
5. **战术层操作**：
   - 累积足够的战术令牌后，点击"召唤棋子"按钮
   - 选择要召唤的棋子类型
   - 在战术棋盘上选择放置位置

## 游戏机制说明

### 战略层（围棋）
- 黑白双方轮流在棋盘交叉点上放置棋子
- 通过包围对方棋子获取领地
- 控制更多领地的一方将获得更多战术令牌

### 战术层（象棋）
- 使用战术令牌召唤棋子
- 不同棋子具有不同的移动规则和战斗能力
- 击败对方棋子可获得战术优势

### 胜利条件
- 消灭对方所有棋子
- 或控制足够多的领地达到胜利阈值
